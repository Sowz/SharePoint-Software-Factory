SPSF Projects

Contains the project templates

Recipes

  • MSBuild Task

    Adds a custom msbuild task to the project.

  • SharePoint 2010 - Application

    Creates a SharePoint 2010 application (farm solution). The application contains an empty SharePoint solution project (.wsp), an optional resources project and a deployment project for deploying all WSPs to multiple farms. SPSF Version 4.1, http://spsf.codeplex.com

  • SharePoint 2013 - Application

    Creates a SharePoint 2013 application (farm solution). The application contains an empty SharePoint solution project (.wsp), an optional resources project and a deployment project for deploying all WSPs to multiple farms. SPSF Version 4.1, http://spsf.codeplex.com

  • Sharepoint 2010 - Solution Package

    Creates a new SharePoint 2010 solution project (.wsp).

  • Sharepoint 2013 - Solution Package

    Creates a new SharePoint 2013 solution project (.wsp).

  • SharePoint Class Library (.dll)

    Creates a new SharePoint class library (.dll). Projects of this type typically contain business logic. The project template generates the correct assembly name, automatically signs the project with the key file of the application and adds the necessary references to SharePoint to the project.

  • SharePoint Solution Deployer Project

    Project template for deployment of SharePoint solution packages. The template can be downloaded seperately from http://spsd.codeplex.com

  • PowerShell Snapin Project
